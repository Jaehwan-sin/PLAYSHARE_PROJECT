## **프로젝트 소개** 
플레이리스트를 들을 때 보통 유튜브나 음악 플랫폼에서 듣게 되면서 
다른 사람들과 의견을 나누며 소통하기 불편하다고 생각해서
개인의 취향이 담긴 플레이리스트를 공유하며 의견도 나누고
추천해주는 사이트를 제작하고자 했습니다!   
   
#### **프로젝트 개발기간**  
**개발 인원** :	1인 <br>
**개발 기간** :	2024년 2월 1일 ~ 2024년 3월 14일
